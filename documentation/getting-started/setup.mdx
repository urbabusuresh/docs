To get started with RaptrOnix, you need to choose a platform to use it on. RaptrOnix is available on the following platforms:

- [Web](#web-app)
- [Desktop](#desktop-app)
- [CLI](#cli)

## Web

To get started with RaptrOnix web client, head to [RaptrOnix.com](http://172.16.111.83:3333) and start using it right away for free.

## Desktop

RaptrOnix Desktop App is a cross-platform desktop application built with Tauri and RaptrOnix Web Client. It is a standalone version of RaptrOnix that can be installed on your computer and used without a browser. RaptrOnix Desktop App is available for Mac, Windows and Linux.

<CardGroup>
  <Card title="Learn more" iconType="light" icon="circle-arrow-right" href="/documentation/clients/desktop"></Card>
  <Card title="Download Desktop App" iconType="light" icon="circle-arrow-right" href="http://172.16.111.83:3333/download"></Card>
</CardGroup>

## CLI

RaptrOnix CLI is the command-line interface for RaptrOnix. It is a standalone version of RaptrOnix that can be installed on your computer and used without a browser. RaptrOnix CLI is available as an npm package.
