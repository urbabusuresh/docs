To get started with wso2apisuite, you need to choose a platform to use it on. wso2apisuite is available on the following platforms:

- [Web](#web-app)
- [Desktop](#desktop-app)
- [CLI](#cli)

## Web

To get started with wso2apisuite web client, head to [wso2apisuite.com](http://172.16.111.83:3333) and start using it right away for free.

## Desktop

wso2apisuite Desktop App is a cross-platform desktop application built with Tauri and wso2apisuite Web Client. It is a standalone version of wso2apisuite that can be installed on your computer and used without a browser. wso2apisuite Desktop App is available for Mac, Windows and Linux.

<CardGroup>
  <Card title="Learn more" iconType="light" icon="circle-arrow-right" href="/documentation/clients/desktop"></Card>
  <Card title="Download Desktop App" iconType="light" icon="circle-arrow-right" href="http://172.16.111.83:3333/download"></Card>
</CardGroup>

## CLI

wso2apisuite CLI is the command-line interface for wso2apisuite. It is a standalone version of wso2apisuite that can be installed on your computer and used without a browser. wso2apisuite CLI is available as an npm package.
