## Getting Started

### Key Features

#### 1. **API Testing**

- Send REST API requests (GET, POST, PUT, DELETE).
- Save requests into **collections** for team sharing.
- Validate responses with built-in assertions.

#### 2. **Test Automation**

- Write test cases in JavaScript/TypeScript.
- Chain requests and validate business logic.
- Integrate with CI/CD pipelines (via CLI).

#### 3. **Scheduled Runs**

- Set up automated test executions (hourly/daily/weekly).
- Receive email alerts for failures.

#### 4. **Allure Reports**

- View detailed test execution summaries.
- Track historical test performance.
